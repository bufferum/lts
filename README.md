## Libraries used

- [gson_2_11.jar](https://github.com/google/gson) - Для работы с json файлами.
- [junit_4_13_2.jar](https://github.com/junit-team/junit4) - Для проведения тестов.
- [junit_hamcrest_core_1_3.jar](https://github.com/junit-team/junit4) - Для проведения тестов.
- [webp_imageio_0_16.jar](https://github.com/sejda-pdf/webp-imageio) - Для преобразования изображений в формат WEBP.
- [javax_mail_1_6_2.jar](https://www.oracle.com/java/technologies/javamail-releases.html) - Для работы с электронной почтой.
- [lombok.jar](https://projectlombok.org) - Эта библиотека предоставляет аннотацию @NonNull которая уменьшает количество проверок и соответственно вызовов Exception.
- [javax_activation_1_1_1.jar](https://www.oracle.com/java/technologies/javamail-releases.html) - Для работы с электронной почтой.
- [mysql_connector_j_8_4_0.jar](https://downloads.mysql.com/archives/c-j/) - Для работы с базой данных
